How can we study using Spaced Practice?

Recalling information directly from memory without any further context. Exactly like flashcards!

What is Retrieval Practice?

While learning be sure to keep questioning why and how and then search for the answers. If possible, make connections with things you already know - the more personal the better.

How can we use Elaboration to study?

It means switching between ideas during study sessions. Don't study a single topic for too long if it feels like learning is harder this way, you are doing it right.

What is meant by the term interleaving?

Link the ideas you are learning to real world examples as often as possible. Use material from the text or search it up online - ask an AI!

How do we learn using concrete examples?

Make sure you encode what you learn through text with visuals and what you learn through visuals with text. There is no preferred learning style but rather try to use all of them to learn effectively.

What is meant by the term Dual Coding?

The 6 best strategies are: spaced practice, retrieval practice, elaboration, interleaving, concrete examples, dual coding.
